# Glossary

| Term | Definition |
|------|------------|
| **CH** | Call Helper — the operational decision support system |
| **Decision Engine** | Core logic layer that parses input, scores confidence, and selects routes |
| **Confidence Score** | Signal strength rating indicating how well an input matches known case patterns |
| **Gray Area Logic** | Handling for ambiguous, conflicting, or low-confidence inputs via structured clarification |
| **Route** | Recommended operational pathway (team, desk, or workflow) for resolution |
| **Case ID** | Identifier used for tracking and pattern recognition across similar issues |
| **Workflow Pathway** | Modular domain flow (registration, activation, qualification, service status) |
| **Agent** | Contact center representative using CH during live calls |
| **Re-evaluation** | Second-pass analysis after clarification signals are incorporated |
