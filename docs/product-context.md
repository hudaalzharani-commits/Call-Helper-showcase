# Product Context

## What is Call Handler (CH)?

**Call Helper (CH)** is an operational decision support system designed for contact center environments.

CH helps frontline agents move from **guessing under pressure** to **structured, validated decisions** during live calls.

## Origin

The project was shaped by direct contact center experience:

- Agents navigating high-volume calls with fragmented knowledge
- Repeated issues without a shared decision framework
- Escalations driven by unclear diagnosis rather than true complexity

CH was built to answer one question:

> How do we help every agent make better decisions on every call?

## Design Philosophy

| Principle | Description |
|-----------|-------------|
| **Decisions, not answers** | CH guides *how* to decide — not just what to say |
| **Validate before deciding** | Low-confidence inputs trigger clarification, not blind routing |
| **Operational realism** | Designed from floor experience, not abstract product theory |
| **Agent-first** | CH supports human judgment; it does not replace agents |

## Target Users

- Contact center agents handling operational and technical inquiries
- Team leads standardizing troubleshooting and routing
- Product and operations stakeholders evaluating decision-system design
