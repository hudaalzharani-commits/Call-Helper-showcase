## Adding Screenshots

1. Take captures at **1440px width** (or 1280px minimum) for consistency
2. Use **light or dark** theme matching the portfolio
3. Avoid exposing internal IDs, credentials, or proprietary backend details
4. Reference images in the root `README.md` using relative paths:

```markdown
![Decision Engine Demo](./screenshots/01-decision-engine-home.png)
```
